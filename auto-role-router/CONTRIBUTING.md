# Contributing to Auto Role Router

Thank you for your interest in contributing! 感谢你的贡献！

## How to contribute / 如何贡献

### Reporting bugs / 报告 bug

If you find a bug, please open an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs. actual behavior
- Your AI assistant and version

如果你发现 bug，请提交 issue 并包含：
- 问题的清晰描述
- 复现步骤
- 期望行为 vs. 实际行为
- 你的 AI 助手及版本

### Suggesting enhancements / 建议改进

We welcome suggestions for:
- New role categories
- Better role assignment logic
- Additional language support
- Integration with other AI assistants

我们欢迎以下建议：
- 新的角色分类
- 更好的角色分配逻辑
- 额外的语言支持
- 与其他 AI 助手的集成

### Pull requests / 提交 PR

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test with at least one AI assistant
5. Update documentation if needed
6. Submit a PR with a clear description

1. Fork 仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 进行修改
4. 至少在一个 AI 助手上测试
5. 如需要，更新文档
6. 提交 PR 并附上清晰的描述

## Code of Conduct / 行为准则

- Be respectful and inclusive
- Focus on constructive feedback
- Help others learn and grow

- 保持尊重和包容
- 专注于建设性反馈
- 帮助他人学习和成长

## Questions? / 有问题？

Feel free to open an issue for any questions or discussions.

欢迎提交 issue 进行任何问题或讨论。
